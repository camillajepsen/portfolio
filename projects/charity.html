Charity Awareness Website


This individual project is a static, awareness-focused website built to inspire and inform users about meaningful ways to contribute to high-impact charities. It was inspired by my interest in charities and my desire to educate myself more about different ways to make an impact.

Developed using HTML, CSS, and JavaScript, the site educates visitors on different donation types, non-financial contributions, and how to evaluate trustworthy charities.

Features include a visually engaging layout with themed background sections, clear content navigation, and mobile-responsive design. The project emphasizes user-friendly information flow, structured HTML semantics, and accessible styling. Hosted on GitHub Pages for public access and version-controlled using Git.
