Real Estate Management System


This individual semester project involves developing a dynamic web-based Real Estate Management System that facilitates property listings, user management, and inquiry handling. Built using PHP, MySQL, HTML, and CSS, the system supports three user roles (Admin, Property Owner, and Customer), enabling property management, user authentication, and secure communication.

Features include a responsive UI, database-driven property search, and validation mechanisms for user inputs. Implemented session handling, CRUD operations, and role-based access to enhance functionality and security.
