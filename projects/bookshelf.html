Online Bookshelf


This personal project is a dynamic web-based Online Bookshelf that helps users organize and track their reading progress. The system is designed inspired by my own love for reading and wanting to keep track of the books I have read and the books I wish to read. The system is built using PHP, MySQL, HTML, and CSS, and allows users to add books, mark them as read, and rate them to create a simple way to manage a personal book collection.
The platform includes a responsive UI for easy navigation, database-driven book storage, and input validation to avoid SQL injections. It supports functionalities like session handling, CRUD operations, and filtering by different parameters to enhance the experience of the system. Currently the system is designed with one user in mind, but I wish to add a login system to support multiple users and add functionality such as book recommendations.
