Finance Manager


This project was done to refresh my Java skills while building a practical command-line system for managing personal finances. It allows users to track their expenses and incomes, manage categories, and save financial data for future reference.
The system is built with Java and uses CSV files to store and retrieve data, ensuring persistence between sessions.

Users can add categories, log transactions with descriptions and dates, and view all financial records in a structured format. I implemented a simple terminal-based menu for easy interaction, and the program automatically loads and saves data. To handle missing files gracefully, the system checks for existing data files and creates them if needed. This project helped me practice file handling, object-oriented programming, and structured data management in Java.
