Journal & Mood Tracker


This project was created to learn about React while building a web application for journaling and mental health. It allows users to write daily journal entries, view past entries on a calendar, and keep track of their moods in an organized way.

The system is built with React for the frontend and Firebase Firestore for storing and retrieving data, ensuring persistence across sessions. It also uses React Router to navigate between the main screens: Journal, Calendar, and Insights.

My next feature to work on is AI-powered insights, which will analyze past entries and moods to provide personalized feedback and trends. I am excited to explore this to learn more about AI integration in applications.
